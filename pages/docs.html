Introduction

Welcome to NexusHub documentation! NexusHub is a modern web platform that brings together essential productivity tools in one beautiful, cohesive interface. All our applications run entirely in your browser, requiring no installation or account creation.

No Account Required

All basic features are available immediately without signing up. Your data stays on your device.

Quick Start

Getting started with NexusHub is simple:

  1. Browse to our Apps page
  2. Click on any available app to launch it
  3. Start using the app immediately - no setup required!
  4. Use the theme toggle to switch between light and dark mode

Whiteboard App

The Whiteboard app is our GoodNotes-style digital canvas for notes, sketches, and annotations.

Tools Available

  • Pen: Draw with precise lines in various colors
  • Highlighter: Semi-transparent highlighting tool
  • Eraser: Remove unwanted marks

Multiple Pages

Create multiple pages using the "+" button in the pages panel. Click on any thumbnail to switch between pages.

Import & Export

Import PDF files to annotate them, or export your work as PNG or PDF format.

Tip

For best PDF import results, use PDFs with simple layouts. Complex PDFs may require additional processing.

Keyboard Shortcuts

Ctrl/Cmd + Z → Undo
Ctrl/Cmd + Shift + Z → Redo

FAQ

Is my data saved?

Currently, drawings are not automatically saved. Make sure to export your work before closing the browser tab.

Can I use NexusHub offline?

Yes! Once the page loads, all apps work completely offline without requiring an internet connection.

Which browsers are supported?

NexusHub works best on modern browsers including Chrome, Firefox, Safari, and Edge.

Troubleshooting

App not loading?

Try refreshing the page or clearing your browser cache.

Drawing feels laggy?

Close other browser tabs to free up memory. Large canvas sizes may impact performance on older devices.

Known Issue

PDF import requires pdf.js library for full functionality. Currently shows a placeholder message.
